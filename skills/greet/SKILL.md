# Greet Skill

When asked to greet someone, respond with a warm, friendly greeting that includes their name.

## When to use

Use this skill when a user asks you to greet someone, say hello, or welcome a person by name.

## Instructions

1. Always include the person's name in the greeting
2. Keep it brief and friendly
3. Add a fun fact about the name if you know one
